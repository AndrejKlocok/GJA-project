Pre spustenia tohto tutoriálu je potrebné mať aj sprovoznené

  • Java8 SDK
  • Maven
  • GlassFish
  • MySql

Tutoriál pre JPA/Hibenate a Primefaces využíva pripojenie do databáze(MySQL5). Pred spustením týchto kapitol tutoriálu je potrebné vytvoriť uživateľa (pripojenie) a databázu podľa konfiguračných súborov persistent.xml a hibernate.cfg.xml. Toto je pripravené v priloženom skripte, v src/dbMysql.sh. Priložený skript inicializuje vytvorí databázu GJA a užívateľa user69 s heslom, sprístupní mu práva na túto databázu. Na tieto operácie je potrebné spúštať tento skript nad root užívateľom mysql(implicitne root ale je možnosť zmeny v skriptu). Následne sa vytvoria tabuľy a vykoná počiatočný seed dát. 


CREATE DATABASE IF NOT EXISTS GJA;
CREATE USER user69@localhost IDENTIFIED BY 'heslo';
GRANT ALL PRIVILEGES ON mydatabase.* TO 'user69'@'localhost';
FLUSH PRIVILEGES;
mysql -uroot -p$ {rootpasswd}  'GJA' < init.sql
