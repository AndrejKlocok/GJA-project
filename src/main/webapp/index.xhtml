Tutorial Java ee aplikácie, ktorá pracuje s Mysql databázou.

Pred spustením tutoriálu je potrebné vytvoriť uživateľa, databázu. Toto je pripravené v priloženom skripte, v src/dbMysql.sh

Priložený skript inicializuje databazu GJA. 


CREATE DATABASE IF NOT EXISTS GJA;
CREATE USER user69@localhost IDENTIFIED BY 'heslo';
GRANT ALL PRIVILEGES ON mydatabase.* TO 'user69'@'localhost';
FLUSH PRIVILEGES;
mysql -uroot -p$ {rootpasswd}  'GJA' < init.sql
