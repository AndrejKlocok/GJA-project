Tutorial Java ee aplikácie, ktorá pracuje s Mysql databázou.

Pred spustením tutoriálu je potrebné vytvoriť uživateľa, databázu. Toto je pripravené v priloženom skripte, v src/resources/db/dbMysql.sh

Tento skript spúšťa jednotlivé query, ktoré vytvoria užívateľa, databázu a tabuľku Student

  • JPA - špecifikácia perzistného API
  • Hibernate - jedna z implementácii JPA
    • Peristencia (načítanie dát z db)
    • Cashing
    • Managovanie vzťahov
    • Mapovanie objektov

Priložený skript inicializuje databazu GJA. 


CREATE USER user69@localhost IDENTIFIED BY 'heslo';
GRANT ALL PRIVILEGES ON mydatabase.* TO 'user69'@'localhost';
FLUSH PRIVILEGES;
