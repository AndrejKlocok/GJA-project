JSF

JavaServer Faces

V tejto časti ukážeme základy JavaServer Faces.

JavaServer Faces (JSF) je framework na tvorbu webových aplikácií, založený na MVC (model-view-controler) architektúre.

Potrebná príprava

3 veci sú potrebné pre vytvorenie jednoduchej stránky pomocou JSF:

  • JDK - napríkad JDK Oracle
  • IDE - napríklad IntelliJ IDEA
  • Aplikačný server - napríklad GlassFish

JSF tagy

jSF definuje niekoľko tagov pre uľahčenie práce. Tu je niekoľko najvýynamnejších s porovnaním od normálnzch HTML tagov. Ďalšie je možno nájsť na tomto odkaze.

HTML tag JSF h Tags
<label> <h:outputText>
<input type="text"> <h:inputText>
<input type="hidden"> <h:inputHidden>
<input type="password"> <h:inputSecret>
<input type="checkbox"> <h:selectBooleanCheckbox>
<input type="radio"> <h:selectOneRadio>
<textarea> <h:inputTextArea>
<select> <h:selectOneListbox>
<select multiple> <h:selectManyListbox>
<img> <h:graphicImage>
